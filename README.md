# Workshop API Tests

## Back End Challenge Execercises (Postman - Newman)

## Setting the project

Clone the repository: `` https://github.com/JesusSalinas/Backend-API-Tests.git `` and install: 

    -  npm install newman
    -  npm install dotenv 
    -  npm install newman-reporter-htmlextra

## Run the project 

Just run int the CLI:
